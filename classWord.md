# الكلاس
هو عبارة عن حاوية كبيرة تحتوي على الكود والمتغيرات والكائنات وهناك أنواع مختلفة من الكلاسات

لتعريف كلاس جديد يكفي فقط كتابة الكلمة class, ثم وضع إسم له, ثم فتح أقواس تحدد بدايته و نهايته. مثال:
</div>

```c#
class Car {

string color = "red";}
```
