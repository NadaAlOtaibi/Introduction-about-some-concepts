# Break

هذه الكلمة تستخدم للخروج من التكرار ويستخدم للفصل بين الحالات والخروج من الحالة في switch cases

مثال:
</div>
```c#
for (int i = 0; i < 10; i++)

{

if (i == 4)

{

break;

}

Console.WriteLine(i);

}
```
